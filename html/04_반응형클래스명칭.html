반응형 클래스 명칭 살펴보기

단위 관련

        px(픽셀) : 고정된 크기의 단위
        %(퍼센트) : 부모 요소이 비례한 상대 단위 
                   기기 크기의 따라 자동으로 웹사이트 내부의 태그 요소들 크기 조정
        em/rem  : 텍스트 크기 기준의 상대단위
        vm/ ㅍh  : 뷰포트 너비 높이의 상대 단위 
                    vh 의 경우 웹사이드에서 세로 크기 단위로 많이 사용

레이아웃 관련

        FlexBOX 1차원 방향으로 정렬할때 사용하는 레이아웃
        Grid 2차원 방향으로 구성할때 사용하는 레이아웃 
        container 콘텐츠를 감싸는 부모 요소로 일반적인 고정 폭 또는 최대 폭 
        Column / Row 그리드 시스템에서 열과 행을 의미 여러 요소를 정렬할
                Colum 의 경우 col 로 주로 사용
                한줄 기준으로 12개의 col 이 존재
            ROW 의 경우 row로 주로 사용 
            가로 정렬 justify-content : center
            새로 정렬 align-items : center
            마진 패딩 
            Margin m-숫자값 영어 사용 
                   m-1 가장 작은 다위의 마진 m-5 가장 큰 단위 마진
            paddig p-5 가장 크게 테두리 안에서 너비를 제공 
            border padding Margin border-radius 1~5 의 숫자로 표현 가능
