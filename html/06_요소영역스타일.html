padding / margin / borber 의 방향성

      (000=== padding, margin, borber 를 나타낸다.)
      000-top : 크기 -> 위쪽 방향크기 지정
      000-button : 크기 -> 아래쪽 방향크기 지정
      000-left : 크기 -> 왼쪽 방향크기 지정
      000-right : 크기 -> 오른쪽 방향크기 지정

      borber 의 속성값은 [크기, 모양, 색상] 형식으로 작성

      parrid | margin | border-width | border-radius : 속성 값(크기)
      
      속성 값 1개 : 상하좌우 하번에 모두 동일한 값을 작성
      속성 값 2개 : 상하좌우 2가지 로 나누어 값을 작성
      속성 값 3개 : 상 좌우 하 (위->좌우->아래쪽) 
      속성 값 4개 : 상 우 하 좌(상 기준으로 시계방향)
