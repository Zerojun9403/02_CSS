CSS(Cascading Style Sheets)

    마크업 언어(HTML) 가 실제로 화면세어 표시돠는 방법을 기술하는 언어 
    (모양, 색상, 크기, 배경이미지, 움직임 )
    하나의 요소에서 여러 CSS코드를 겹쳐서 원하는 스타일을 만들어냄
    w3c에서 css를 웹표준으로 지정합

CSS 선택자

        현재 문자 내에서 특정요소를 선택하는 문자 또는 문자얄(기호)

        * 요소(Element) : 시작태그 + 속성 + 내용 + 종료태그
    ** css 코드를 작성하는 방법**
  • head 태그 안에 style 태그르르 작성한 후 css 코드 작성
  • 원하는 요소에 직접 style 속성을 이용해서 css 코드 작성
  • 외부 .css 파일을 연결하는 방법 (현업에서 많이 사용)

태그 선택자

        - 현재 문서 내에 특정 태그를 모두 찾아 선택하는 선택자
        [작성법]
        
            태그명 {css코드;}

id(#) 선택자

        html 문서 내에서 
        id가 일치하는 요소를 선택하는 선택자
[작성법] #id속성값{css 코드;}

class(.) 선택자

    [작성법]
    
        .class속성값{css 코드 ;}
    
    *** 요소에서 css 를 적용하기 위한 선택자를 작성 할때 id 선택자 보다 class 선택자를 이용하는것이 좋다!!!

모든 요소 선택자(*, asterRisk)

단독 사용시 : 문서에 존재하는 모든 요소 선택자
    다른 선택자와 환용 사용시 : 선택된 요소의 모든 자식/후손/형제 까지 모두 적용
    CSS 적용 우선순위
    1. 선택자 범위가 가장 작은 선택자가 우선순위를 갖는다. 
    (!important > 안라인 스타일> 아이디> 클래스 > 태그 > 모든요소)
    2. 태그내에서 같은 속성값이 있을 경우 가장마지막에 작성된 클래스 내에 존재하는 css 속성값이 적용
